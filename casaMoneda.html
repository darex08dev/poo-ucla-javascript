Casa de Moneda - ¿Cómo funciona?

La Casa de la Moneda cuenta con una cantidad de billetes disponibles. El usuario puede agregar un cliente con su ID, el valor del billete a cambiar (solo admite $20, $50 y $100). Y elegir la cantidad de billetes de $10, $20 y $50 que quiere recibir a cambio.

El programa tiene una cantidad de billetes iniciales (15 billetes de cada tipo). El usuario debe ingresar la cantidad correcta de billetes a recibir, de hacer lo contrario, el programa le indicará que la operación en inválida. Igualmente si la cantidad de billetes solicitados no está disponible.

El Usuario puede eliminar y agregar cuantos clientes quiera. Para agregar solo debe precionar y el programa le pedirá los datos para registrar al cliente. El programa no permite tener dos clientes con el mismo ID. Para eliminar un cliente debe precionar y proporcionar el id del mismo. Si el ID no está registrado, el programa le dirá que no existe.
